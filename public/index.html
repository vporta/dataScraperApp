KICKSTARTER SCRAPER

Kickstarter Scraper is a company specializing in data extraction from Kickstarter web pages.

Overview

This app allows you to scrape the Kickstarter page's featured project for the music category.

You can also save notes for each scraped project by clicking on the text. Your note will be automatically saved to refer back to anytime.

Directions

  1. Type 'scrape' in the url bar
  2. Hover over and click on the scraped text
  3. Type in the input box
  4. Click submit to save your note

Classic Note Saving

Feel free to refer back to your note by clicking on the scraped text.

Your note will always appear inside the input box.
