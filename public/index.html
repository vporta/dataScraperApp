KICKSTARTER SCRAPER

Kickstarter Scraper is a website specializing in data extraction from Kickstarter web pages.

Overview

This app allows you to scrape Kickstarter's featured projects. Stay up to date on the featured projects of the day.

You can also save notes for each scraped project by clicking on the text. Your note will be automatically saved.

Directions

  1. Type in one of the Scrape commands in the url bar
  2. Refresh the page
  3. Hover over and click on the scraped text
  4. Type in the input box
  5. Save your note

Classic Note Saving

Feel free to refer back to your note by clicking on the scraped text.

Your note will always appear inside the input box.
